JotLoader Administration

General
Die Administration von JotLoader erfolgt im Backend und bietet folgende Möglichkeiten:
  • Verwaltung der Downloaddateien
  • Verwaltung der Kategorien
  • einstellbares Download-Verzeichnis für die Dateien, die als Download bereitgestellt werden (mit wählbarer "Auto Detect" Funktion)
  • HTML-Definition der Darstellung von Kategorien und Dateitabellen im Frontend (innerhalb der Vorlagetexte ist es möglich, vordefinierte Tags zu benutzen)
  • Konfiguration und Auswertung der Download-Statistik

Hier muss man bemerken dass Download-Dateien erst im Frontend erscheinen, wenn sie einer Kategorie zugeordnet sind. Eine Statusänderung auf "veröffentlicht" in der Dateiliste reicht also nicht aus.

Kategorien
In der Kategorien-Manager können Sie die Formatierung von Kategorie Beschreibung (einschließlich der Einbettung von Bildern) wie üblich für HTML-Editor durchführen. Die Format-Einstellung für Kategorie Titel ist in JotLoader Einstellungen / Dateien Header-Template möglich.

Spracheinstellung
JotLoader bittet die Möglichkeit von zwei grundlegende Sprache Sets - Englisch (en-GB) oder Deutsch (de-DE) zu wählen. Für die Verwendung der deutschen Sprache ist es notwendig, deutsche Installation von Joomla (z.B. http://www.joomla-grundlagen.de/downloads-joomla-1.5.x-versionen) vor der Installation von JotLoader durchzuführen.

Nach der Installation können Sie die Sprache im Administrator-Erweiterungen-> Sprach-Manager einsetzen. Mit beschriebenem Vorgehen wird automatisch auch die Sprache-Einstellung für JotLoader Komponente verändert.

Für Website-Frontend können Sie jede mögliche Sprache benutzen (in UTF-8-Codierung) dank der richtigen Vorlage-Einstellungen in JotLoader Administrator. Nur Bug-Tracker Funktionen und Tasten bleiben in der englischen Sprache eingestellt (einfach weglassen Bug Tracker, wenn Sie möchten).

Installation / Uninstallation
Hochladen und installieren Sie die Komponente über JotLoader Joomla Administrator mit dem normalen Verfahren (Erweiterungen-> Install/Uninstall-> Durchsuchen / Datei hochladen & Installieren).

Richten Sie Ihre Vorzüge und die Einstellungen für diese Komponente (Komponenten-> JotLoader-> Einstellungen, sehe JotLoader Einstellungen) aus. Dann wählen Sie in Joomla Administrator richtige Menü auf, um den Inhalt von JotLoader zum bestimmten Menüeintrag einzuknüpfen (sehe JotLoader Menü).

Für die Deinstallation von JotLoader Komponente wählen Sie bitte Joomla Administrator-Erweiterungen-> Install/Uninstall-> Komponenten Link und legen Sie JotLoader Radio-Auswahl auf "ON", und drücken Sie dann Deinstallieren-Button in der Symbolleiste. Datenbank-Tabellen von JotLoader bleibt intakt mit der zuletzt gespeicherte Inhalt.

Upgrade von alten Version
Deinstallieren Sie die alte Version im Erweiterungen-> Installieren / Deinstallieren Auswahl Link "Komponenten" und setzen Sie JotLoader Radio-Auswahl auf "ON", und drücken Sie dann Deinstallieren-Button in der Symbolleiste. Datenbank-Tabellen von JotLoader bleibt intakt mit der zuletzt gespeicherte Inhalt. Installieren Sie neue Version von JotLoader wie oben beschrieben.
