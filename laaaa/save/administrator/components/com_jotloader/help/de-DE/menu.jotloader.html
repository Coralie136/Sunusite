JotLoader Menü Zuordnung

Die JotLoader Komponente könnte auf verschiedene Weise dem Menü zugeordnet sein. Sie können auswählen entweder alle Kategorien oder eine bestimmte Kategorie allein auf der Front-End Seite darzustellen. Diese Zuordnung müssen Sie während Erstellung von neuem Menüeintrag auf der Eingabefläche Basisparameter durchführen.

Grundeinstellungen ermöglichen alle zugelassene Kategorien mit Default-Parametern (eingestellt in JotLoader Einstellungen) auf eine einzige Front-End-Seite abbilden.

Fortgeschrittene Administratoren können eine bestimmte Kategorie für die Front-End Presentation auszuwählen und dabei Default-Parametern oder spezifischen Layout für Kopfzeile und Dateien (eingestellt in der Eingabefläche Komponentenparameter) benutzen.

Neuer Menüeintrag
Wählen Sie die Menü nach Ihre Wahl in Administrator / Menüs (z.B. Haupt-Menü) und drücken Sie dann Schaltfläche NEU in der Symbolleiste.

Im Baum Menütyp wählen klicken Sie auf JotLoader und dann auf Categories Presentation Link. Der neue Menüeintrag Definition Seite wird geöffnet. Hier ist erforderlich, Menüeintrag Titel und Alias eingeben. Wenn Standard-Layout ist zufriedenstellend reichts nur Schaltfläche "Speichern" drücken in der Symbolleiste und neuen Menüeintrag ist erstellt.

Für benutzerdefinierte Layout können Sie wählen :

Basisparameter

JotLoader Kategorie
Auswahl der Option für die Darstellung von aller Kategorien oder eine bestimmte Kategorie (nur die Kategorien eingestellt in JotLoader Kategorie Manager sind abgebildet).
Formatierung der Kopfzeile
Wählen Sie die Option "global" für Kopfzeile Standard-Layout (wie in JotLoader Einstellungen eingestellt ist) oder Option "menuspezifisch" für Format die in Eingabefeld Header von Komponentenparameter definiert ist.
Formatierung der Dateizeile
Wählen Sie die Option "global" für Dateien Standard-Layout (wie in JotLoader Einstellungen eingestellt ist) oder Option "menuspezifisch" für Format die in Eingabefeld Dateien von Komponentenparameter definiert ist.

Komponentenparameter
ist notwendig einzustellen nur im Falle von Nicht-Standard-Einstellungen im Basisparameter. Die Definitionen für Dateien und die Kopfzeile sind die gleichen wie in JotLoader Einstellungen.
