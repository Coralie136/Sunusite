JotLoader Administration

General
Management of files categories and download files is performed in administration back-end in the standart manner.

Please take care that the download files are shown in front-end only in the case, when they are assigned to the existing category. The file status "published" in the File List table is therefore not sufficient to display file in the front-end.

Categories
In Categories Manager you can set the formatting of category description text area (including embedding of pictures) as usual for this type of editor. The format setting of category title is performed in JotLoader Settings Files Header template.

Language setting
JotLoader has possibility for selecting two basic language sets - English (en-GB) or German (de-DE). For using German language set is necessary to have German installation of Joomla (e.g. http://www.joomla-grundlagen.de/downloads-joomla-1.5.x-versionen/joomla-v.1.5.2-komplett-deutsch.html) before installation of JotLoader.

After the installation you can select language sets in Administrator Extensions->Language Manager which you like to prefere as default (take care for Site/Administrator tabs to select which part of Joomla presentation you like to change). With described change is automatically changed also language set for JotLoader component.

For site front-end you can use any possible language set (included in UTF-8 coding) for your presentation of download files with proper JotLoader settings in Administrator. Only Bug Tracker functions & buttons shall remain in basic language set (easy to omit Bug Tracker when you like).

Installation / Uninstallation
Upload and install the JotLoader component via Joomla Administrator using the normal procedure (Extensions->Install/Uninstall->Browse/Upload File & Install).

Set up your preferencies and settings for this component (Components->JotLoader->Settings, see JotLoader Settings). Then select in Joomla Administrator proper menu set to which you like to assign the JotLoader content (usually is selected Menu->mainmenu, see JotLoader Menu).

For uninstallation of JotLoader component please select via Joomla Administrator Extensions->Install/Uninstall->Components link and set JotLoader radio selection to "ON", then press Uninstall button in the toolbar. Database tables of JotLoader will remain intact with the last saved content.

Upgrade from old version
Uninstall the old version in Extensions->Install/Uninstall selecting link "Components" and set JotLoader radio selection to "ON", then press Uninstall button in the toolbar. Database tables of JotLoader will remain intact with the last saved content. Install new version of JotLoader as described above.
