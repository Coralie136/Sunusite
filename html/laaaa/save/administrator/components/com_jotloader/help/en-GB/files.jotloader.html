File Management

List Files

This page gives overview about files and their attributes which are stored in the component database. Files which are stored only in the database and no more existing in the file system are marked with red color in the File Name column (tooltip displays stored path to the file).

Column titles in blue color have order functionality. Clicking here on the column title you can change order of presented files. When you like to apply the selected order to the front-end presentation then click to small store icon next to the rigth end of the column title. Because frontend presentation is based on categories you can see the proper frontend ordering after pressing the Show Order button on the toolbar (when this button is not displayed on the toolbar then the diplayed order match the order of displayed files in the frontend).

Move Files

Here it is possible to move the entire selected group of files to the new target category. In the case when the files names are marked with red color (meaning - these files are not readable any more on the given upload path in the file system) moving these files to selected category also move the files to the new upload path set in the Settings.

Delete Files

Here is difference against JotLoader component, ver.1.x, because the Delete action after warning is deleting files not only from database but also from the file system. When you do not like to loose these files then you have to save them before Delete action.
