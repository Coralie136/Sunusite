JotLoader Menu Assignment

Here is a possibility of multiple assignments of chosen categories to the selected menu. During creating the new menu item you can select in Parameters - Basic how to present download categories and files. Default settings allows to use default parameters (set in JotLoader Settings) for all categories on one single front-end page.

Advanced administrators can select particular category for front-end presentation and/or the usage of specific layouts for files header / files as they are specified in Parameters - Component pane.

New Menu Item
Select the chosen menu in Administrator/Menus menu (e.g. Main Menu) and then press NEW button on the toolbar.

In the Select Menu Item Type tree click on JotLoader and then on Categories Presentation link. The new menu item definition page is opened. Here is neccessary to set menu item title and alias. When you like standart default layout here is enough to press SAVE button on the toolbar and new menu item is created.

For custom layout you can choose :

Parameters - Basic

JotLoader Category
Select option of all categories or one specific category (shown are categories set in Category Manager of JotLoader).
Header Format
Select option default layout (as set in JotLoader Settings) or select menu item related format with adjustment of Header definition in Parameters - Component pane.
Files Format
Select option default layout (as set in JotLoader Settings) or select menu item related format with adjustment of Files definition in Parameters - Component pane.

Parameters - Component
is necessary to setup only for non-default settings in Parameters - Basic pane. The definitions for Files Header and Files are the same as described in JotLoader Settings.
