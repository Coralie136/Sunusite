JotLoader Einstellungen

Globale Einstellungen
In diesem Teil des Eingabeformulars es ist möglich folgende Parameter einzustellen :

Datum/Uhrzeit-Anzeige
Datum und Zeit Format, welche ist automatisch bei der Anerkennung der neuen Datei im Upload-Verzeichnis eingestellt (AutoDetect Flag muss auf ON gesetzt).

Das Datum/Uhrzeit-Format der Daten, die bereits in der Datenbank gespeichert sind, wird nach der Änderung dieses Formats unverändert bleiben.

Upload-Verzeichnis
Standard Download-Verzeichnis für JotLoader ist während der Installation auf "download" gesetzt. Diese Einstellung ist möglich später in einem anderen Verzeichnis-Namen ändern (vor der Nutzung müssen Sie aber manuell das gewünschte Verzeichnis in dem Server-Dateisystem erstellen).

Wenn "Automatische Dateierkennung"-Flag eingesetzt wird, dann alle neue Dateien in diesem Verzeichnis werden in Dateien-Manager automatisch eingeholt (mit dem Status UNVERÖFFENTLICHT).

Achten Sie darauf, für die automatisch ermittelten Dateien die Jot Loader Kategorien einzustellen, was für die ordnungsgemäße Darstellung von Download-Dateien im Front-End notwendig ist.

Automatische Dateierkennung
Wenn "Automatische Dateierkennung"-Flag eingesetzt wird, dann alle neue Dateien in diesem Verzeichnis werden in Dateien-Manager automatisch eingeholt (mit dem Status UNVERÖFFENTLICHT).

Achten Sie darauf, für die automatisch ermittelten Dateien die Jot Loader Kategorien einzustellen, was für die ordnungsgemäße Darstellung von Download-Dateien im Front-End notwendig ist.

Bitte nehmen Sie in Betracht diese "Automatische Dateierkennung"-Flag Einstellung, wenn Sie möchten Download-Datei aus Datei-Liste löschen . Wenn "Automatische Dateierkennung"-Flag gesetzt ist, dann die Schaltfläche "Löschen" auf der Dateimanager Seite dient nur für das Entfernen der Datei aus der Datenbanktabelle. Nach dem aber Automatische Erkennung findet die Download-Datei im Download-Verzeichnis auf dem Server und Datei (automatisch erkannt) ist wieder in die Datenbanktabelle zurückgeschrieben. In diesem Fall müssen Sie vor dem Löschen auch die Datei von dem Server Download-Verzeichnis entfernen(in der Einstellungen für die gegebenen Datei - Link unterhalb des Datei-Speicherort).

Download Statistik aktivieren
Einfache Statistik des Datei-Herunterladen wird von JotLoader-Komponente durchgeführt, wenn dieses Flag gesetzt ist.

Der Zugang zu dieser Statistik ist über Systemsteuerung / Download Statistik möglich.

Vorlagen
Definitionen in diesen Vorlagen ermöglichen die graphische Zuordnung von Download-Kategorien und -Dateien zur ihre Front-End-Präsentation. Hier besteht die Möglichkeit, verschiedenen Layouts für die Download Kategorien während der JotLoader Zuordnung zu bestimmten Menüpunkt (siehe JotLoader Menü) einzustellen.

Die Vorlagenformatierung ermöglicht eine individuelle Präsentation von JotLoader Front-End-Seiten (verschiedene Schriftarten, Farben, Sprachen) nach HTML-Standards. Aber wenn Sie das Layout ändern, es ist gut zu praktizieren eine Kotrolle für die Front-End-Präsentation bei den in der Regel häufig verwendete Browser-Typen (Firefox, Internet Explorer, Opera etc.) durchzuführen.

Zu jeder Kategorie hier ist möglich, ausgewählte Bilder durch seine Einbettung in die Kategorie Beschreibung (siehe JotLoader Administration) zulegen.

Wir haben Möglichkeit Layout für drei wichtigsten Teile der Präsentation: Kategorie, Header-Dateien und Dateien zu bestimmen. Grundlegende Struktur für Gesamt-Layout ist folgende:

Kategorie XXXXXXXXXXX
  Kopfzeile xxxxxxxxxxx
    Dateien ........
           ........

Kategorien
Hier ist default Standard-Präsentation der Download Kategorien definiert.

Sie können die Präsentation von jeder Kategorie während ihrer Zuordnung zu Menüs (siehe JotLoader Menü) separat ändern.

Dateien Überschrift
Es wird hauptsächlich für fix-definierte Texte der Tabellekopfzeile benutzt. Bei längeren Download-Dateien Titeln ist es möglich, größere Breite von Datei-Titel Spalte einzustellen z.B. width = 40% (ähnliche Veränderung muss man auch für Dateienvorlage durchführen).
Dateien
Diese Vorlage enthält Basis-Definition von dynamisch generierten Teil der Dateien-Präsentation. {} Tags sind spezialle JotLoader Tags, die automatisch mit der Datenbankinhalt während der Frontend-Seite Präsentation ersetzt sind.
{file_id}
(integer) interne Identifikationsnummer der Download-Datei in der Datenbank
{file_title} {version} {license_type} {cat_id} {size} {date_added} {url_home} {author} {url_author}
Tags sind gefüllt mit Download-Datei-Parametern.
{url_download}
benutzt für Download-Link auf der Front-End-Seite
{url_bugs}
benutzt für Bug-Tracker-Link auf der Front-End-Seite
{downloads}
(integer) Zähler von durchgeführten Download-Ausrufe
