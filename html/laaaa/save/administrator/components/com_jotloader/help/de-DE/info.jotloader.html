Bemerkung für beliebige Version von JotLoader Komponente :
JotLoader war ursprünglich basierend auf doQment Komponente, ver.1.0.0 beta, von pimpmyjoomla.com in Mai 2007 unter GNU/GPL Lizenz entwickelt. JotLoader, Ver.2 ist voll überbearbeitete Komponente für Joomla 1.5.x mit vielen neuen Funktionen. Die Komponente ermöglicht eine Darstellung von Dateien im Frontend in der beliebigen Einordnung, Funktionen für Problembeschreibung, einfache Download-Statistik sowie auch die Autopublishing von hochgeladenen Dateien.

JotLoader Name:
In Websters Wörterbuch hat das Substantiv "jot (Pünktchen)" folgende Bedeutung :
1. Eine kurze (und eilig handgeschriebene) Notiz.
2. Ein kleiner, aber spürbarer Betrag.

Der Name JotLoader soll die Fähigkeit der Downloader-Komponente, mit kurzen Kommentaren (Mitteilungen) zu den einzelnen Download-Link zu arbeiten, ausdrücken.
